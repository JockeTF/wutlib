# Wutlib

Demonstrates something floofy.

```
$ python3 -m wutlib
Floof is floofy!
```

```
$ python3 wutlib/__main__.py
Floof is floofy!
```

```
$ python3 -m unittest --quiet
Floof is floofy!
----------------------------------------------------------------------
Ran 3 tests in 0.000s

OK
```

```
$ poetry install --quiet
$ poetry run wutlib --help
This isn't an argument clinic!
```

```
$ pip install --quiet .
$ wutlib --help
This isn't an argument clinic!
```
